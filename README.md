bubbl3tea
=========

bubbl3tea personal website.

Current Features
----------------

* Simple blog:
  * Add/edit/delete posts.
  * Friendly IDs for slugging.
  * Clearance for authentication.
  * Kaminari for pagination.
  * Media hosted via Wistia.
* High Voltage static pages for everything else.
